# FlowDesk

Sistema desktop de gestão comercial desenvolvido em Java 21, com foco em
desempenho, usabilidade e organização.

## Sobre o projeto

O FlowDesk é um ERP Desktop para Windows inicialmente desenvolvido para
um estúdio de tatuagem e piercing, com arquitetura preparada para
expansão para outros segmentos.

## Objetivos

-   Centralizar operações comerciais.
-   Controlar vendas, estoque e financeiro.
-   Oferecer uma interface moderna e intuitiva.
-   Criar uma base escalável para novos módulos.

## Principais funcionalidades

-   Caixa (PDV)
-   Controle de estoque
-   Gestão financeira
-   Registro de serviços de tatuagem
-   Controle de comissão de tatuadores
-   Dashboard gerencial

## Tecnologias

-   Java 21
-   Spring Boot 3
-   JavaFX
-   PostgreSQL
-   Maven
-   JUnit 5
