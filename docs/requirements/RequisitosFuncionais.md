# Requisitos Funcionais - FlowDesk

## Caixa

RF001 - O sistema deve permitir registrar vendas de produtos.

RF002 - O sistema deve permitir registrar serviços.

RF003 - O sistema deve aceitar pagamentos em dinheiro, Pix, crédito e
débito.

RF004 - O sistema deve permitir pagamentos combinados.

RF005 - O sistema deve permitir descontos por percentual ou valor fixo.

RF006 - O sistema deve permitir cancelamento de vendas com registro.

## Estoque

RF007 - O sistema deve permitir cadastro de produtos.

RF008 - O sistema deve controlar quantidade em estoque.

RF009 - O sistema deve realizar baixa automática após venda.

RF010 - O sistema deve alertar estoque mínimo.

## Financeiro

RF011 - O sistema deve controlar fluxo de caixa.

RF012 - O sistema deve registrar sinais de tatuagem.

RF013 - O sistema deve calcular comissão de tatuadores.

## Dashboard

RF014 - O sistema deve apresentar indicadores de vendas.

RF015 - O sistema deve apresentar informações financeiras.

Versão: 0.1.0
