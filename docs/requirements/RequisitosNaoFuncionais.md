# Requisitos Não Funcionais - FlowDesk

## Desempenho

O sistema deve possuir resposta rápida nas operações comuns.

## Disponibilidade

O sistema deve funcionar sem dependência obrigatória de internet.

## Segurança

O acesso ao sistema deve exigir autenticação.

## Manutenibilidade

O código deve seguir boas práticas:

-   SOLID
-   Clean Code
-   Arquitetura modular

## Compatibilidade

O sistema deve ser executado em Windows.

## Banco de Dados

O sistema utilizará PostgreSQL.

Versão: 0.1.0
