# Visão Geral - FlowDesk

## Identificação

Nome do projeto: FlowDesk

Tipo: Sistema ERP Desktop

Plataforma: Windows

## Descrição

O FlowDesk é um sistema de gestão comercial desenvolvido para pequenos
negócios, inicialmente focado em estúdios de tatuagem e piercing.

O sistema tem como objetivo simplificar processos administrativos,
oferecendo controle de vendas, estoque, financeiro e serviços.

## Público inicial

Estúdios de tatuagem e piercing.

## Possível expansão

A arquitetura permitirá adaptação para outros segmentos comerciais.

## Premissas

-   Funcionamento offline.
-   Interface moderna.
-   Alta performance.
-   Facilidade de utilização.

## Versão

0.1.0
